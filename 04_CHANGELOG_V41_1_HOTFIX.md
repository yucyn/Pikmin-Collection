# Pikmin Collection - V41.1 HOTFIX 變更紀錄

## 版本資訊
- **版本號**: V41.1 (HOTFIX_GPS_COUNTRY_IMAGE_TAG)
- **更新日期**: 2026-05-11
- **主要目標**: 修正 GPS 國家判定衝突、實作高精度圖像自動標籤系統。

---

## 🛠️ 修正功能 (Features Fixed)

### 1. 全域 GPS 國家判定系統 (Global Country Resolution)
- **解決國界重疊問題**: 重新調整歐洲與美洲國家的判定權重（如：奧地利/瑞士、葡萄牙/西班牙、芬蘭/挪威），解決邊界區域誤判的問題。
- **在地化支援**: 在關鍵字判定中補齊了繁簡體中文（如「意大利/義大利」）與歐系國家母語（如「Deutschland」）。
- **新增國家**: 正式支援 **塞浦路斯 (Cyprus)** 與 **密克羅尼西亞 (Micronesia)**。
- **API 健壯性**: 引入 5 秒超時保護，當 OSM API 無回應時自動降級至精確的離線座標規則判定。

### 2. 智慧型圖像自動標籤 (Heuristic Image Tagging V2)
- **全通路偵測**: 支援檔案上傳、剪貼簿貼上、網址拖曳三種入口的自動判定。
- **花朵判定 (Flower)**: 精確偵測右下角具備「厚白邊貼紙感」的大型花朵圖案。
- **蘑菇判定 (Mushroom)**: 嚴格執行「三位一體」判定——必須同時偵測到「綠色 CLEARED! 蓋章」與「黃色星星/點數文字」。
- **排除干擾**: 透過嚴格的 RGB 比例過濾與區域鎖定，成功排除了「復活節卡片」淡綠色背景與右上角 Logo 的誤報。
- **隱藏判定 (Hidden)**: 當非花朵或蘑菇時，自動歸類為一般明信片。

## 🚫 未變更項目 (What was NOT modified)

- **資料庫結構**: 本次更新未更動 Firebase 資料庫結構，舊有資料可完美相容。
- **圖片壓縮核心**: 圖片壓縮流程保持不變，標籤辨識是在壓縮後的數據上進行。
- **地圖顯示邏輯**: 除了更新國家過濾清單外，地圖標記點的顯示方式保持原樣。
