# Pikmin Collection - V41.1 HOTFIX 變更紀錄

## 版本資訊
- **版本號**: V41.1 (HOTFIX_GPS_COUNTRY_IMAGE_TAG)
- **更新日期**: 2026-05-11
- **主要目標**: 修正 GPS 國家判定衝突、實作高精度圖像自動標籤系統。

---

## 🛠️ 修正功能 (Features Fixed)

### 1. 全域 GPS 國家判定系統 (Global Country Resolution)
- **解決國界重疊問題**: 重新調整歐洲與美洲國家的判定權重（如：奧地利/瑞士、葡萄牙/西班牙、芬蘭/挪威），解決邊際區域誤判的問題。
- **在地化支援**: 在關鍵字判定中補齊了繁簡體中文（如「意大利/義大利」）與歐系國家母語（如「Deutschland」）。
- **新增國家**: 正式支援 **塞浦路斯 (Cyprus)** 與 **密克羅尼西亞 (Micronesia)**。
- **API 健壯性**: 引入 5 秒超時保護，當 OSM API 無回應時自動降級至精確的離線座標規則判定。

### 2. 智慧型圖像自動標籤 (Heuristic Image Tagging V2) - 深度優化版
- **大花偵測 (Flower)**: 採用「廣域掃描 + 高亮度白邊偵測」，確保不同位置的大花標籤都能被精準捕捉。
- **蘑菇偵測 (Mushroom) - 超感度模式**:
    *   **三位一體判定**: 必須同時偵測到「黃色星星」、「黑色文字」與「綠色蓋章」。
    *   **抗背景干擾**: 放寬了 RGB 頻道限制，能有效識別在「藍色花海」等複雜背景下的綠色蓋章。
    *   **垂直全掃描**: 掃描區域擴大至 Y=400（直達底部），適應各種位置的資訊方塊。
    *   **極低門檻觸發**: 調優像素門檻，確保 1-2 顆星或縮圖模糊的卡片也能正確辨識。
- **排除誤判**: 成功排除復活節活動背景、貓咪照片、門口公告等非標籤元素的干擾。
